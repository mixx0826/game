📤 上传新游戏

请上传包含 index.html 的 ZIP 文件，最大10MB
支持 JPG、PNG、GIF 格式，建议尺寸 300x200
上传指南
  • 游戏文件必须是 ZIP 格式
  • ZIP 文件内必须包含 index.html
  • 支持标准的 HTML5 游戏
  • 文件大小限制：10MB
  • 建议提供游戏缩略图
重要提醒

由于 Vercel 无服务器环境限制，文件上传功能在当前部署中不可用。 建议使用外部云存储服务或部署到支持文件存储的平台。

⚙️ 系统设置

平台信息
平台名称： 乐玩小游戏平台
版本： 1.0.0
部署环境： Vercel 无服务器
数据库： SQLite (内存模式)
技术栈： Node.js + Express.js
部署限制
Vercel 无服务器限制：
  • 使用内存数据库，重启后数据丢失
  • 无法持久化存储上传的文件
  • 请求执行时间限制
  • 冷启动可能导致响应延迟
生产环境建议：
  • 使用外部数据库 (PlanetScale, MongoDB Atlas)
  • 使用云存储服务 (AWS S3, 阿里云OSS)
  • 考虑部署到 VPS 或云服务器
